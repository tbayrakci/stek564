Large Language Models (LLMs) were utilized during this project as a pair-programming and debugging assistant. Specifically, AI tools were used to assist in structuring the PyTorch boilerplate code for the Neural Networks, conceptualizing the Bradley-Terry Preference Reward formatting, and formatting the final markdown report. All implemented algorithms (CQL, IQL, CMDP Ablations) were manually verified, parameterized, and theoretically justified by the author.
